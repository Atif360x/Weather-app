# 🌤️ Weather App

A clean and minimal weather app built with HTML, CSS, and JavaScript.
Features dynamic backgrounds that change based on real-time weather conditions and time of day.

## ✨ Features

- 🔍 Search any city in the world
- 🌡️ Real-time temperature & feels like
- 💧 Humidity & wind speed
- 🌄 Dynamic background based on weather condition
- 🌙 Day & Night background switching
- ❌ Error handling for invalid cities

## 🛠️ Built With

- HTML
- CSS (Glassmorphism UI)
- JavaScript (Vanilla)
- OpenWeatherMap API
- Google Fonts (Nunito + Material Symbols)
